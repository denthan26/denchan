Wins上使用ffmpeg录屏并设置编码格式

  • By Denchan
  • November 17, 2024

放弃其他软件，在Windows上使用ffmpeg进行屏幕录制。

版本：ffmpeg-N-117688-gd6b2d08fc7-win64-gpl-shared（文件夹的名称是这个）

屏幕及音频录制： 


// 获取已有设备，如果没有，则需要其他软件包
ffmpeg -list_devices true -f dshow -i dummy
// 将音频设备填入，desktop 表示录制整个桌面
ffmpeg -f gdigrab -i desktop -video_size 1920x1080 -framerate 30 -f dshow -i audio="麦克风阵列 (适用于数字麦克风的英特尔® 智音技术)" -c:v libx264 -c:a aac output.mp4

// 重新编码，wins自带的媒体播放器不支持第二步的编码设置
ffmpeg -i input.mp4 -vcodec libx264 -pix_fmt yuv420p -acodec aac output.mp4
